# liquidsoap-radioplus-and-EMF

I have built this because I wanted to ensure a full professional stream with regards relaying Radio Plus.

Radio Plus has been using HLS but has failed to actually insert their song and artist titles, so this project allowed me to change transcode their existing stream and make a custom song title. 

I also wanted to bring K-Love and Air 1 back into Icecast feeds by converting their HLS feeds.

This tutorial shows you how to do exactly that.
